# История коммитов в Git

История коммитов — это **полная летопись проекта**.  
Git позволяет удобно просматривать, фильтровать и анализировать изменения.

---

## 📜 Просмотр истории — `git log`

Показывает список коммитов (от новых к старым):

```bash
git log

📌 Каждый коммит содержит:

SHA-1 хеш (уникальный ID)
автора и email
дату и время
сообщение
📊 Форматы вывода
Компактный список:
git log --oneline
Граф веток:
git log --graph --oneline
Статистика изменений:
git log --stat
Полный diff каждого коммита:
git log -p
🔎 Фильтрация истории
# Последние N коммитов
git log -3

# За период
git log --since="2 weeks ago"
git log --since="2024-03-01" --until="2024-03-15"

# По автору
git log --author="Ivan"

# По сообщению
git log --grep="fix"

# По файлу
git log -- README.md

# По функции
git log -L :functionName:file.js
⚙️ Кастомный формат
git log --pretty=format:"%h - %an, %ar : %s"

📌 Полезные плейсхолдеры:

%h — короткий хеш
%an — автор
%ar — относительная дата
%s — сообщение
🧭 Навигация по коммитам
Основные ссылки:
HEAD — текущий коммит
branch — последний коммит ветки
hash — конкретный коммит
tag — именованная точка
Относительные ссылки:
HEAD~1   # на 1 коммит назад
HEAD~2   # на 2 назад

HEAD^    # родитель
HEAD^2   # второй родитель (merge)

master~3 # 3 коммита назад от ветки

📌 Можно комбинировать:

HEAD~2^2
🔍 Просмотр коммита — git show
# Последний коммит
git show

# Конкретный
git show a5f3b8c

# Несколько шагов назад
git show HEAD~3
Полезные варианты:
git show --stat        # только статистика
git show --name-only   # только файлы
🔎 Поиск в истории
# Найти изменение строки
git log -S "function authenticate"

# По regex
git log -G "authenticate.*password"

# Кто изменял строки
git blame file.js
🌳 Визуализация истории
git log --graph --oneline --all

Расширенный вариант:

git log --graph --pretty=format:'%h -%d %s (%cr) <%an>' --all
🔄 Сравнение коммитов
# Между двумя коммитами
git diff HEAD~2 HEAD

# С текущим состоянием
git diff a5f3b8c

# Только список файлов
git diff --name-only HEAD~5 HEAD
🧪 Практические сценарии
🔍 Поиск багов
анализ истории файла
поиск коммита, где появилась ошибка
📈 Анализ проекта
git shortlog -sn   # вклад по авторам
🚀 Подготовка релиза
git log v1.0..HEAD --oneline
📅 Активность по дням
git log --format="%ad" --date=format:"%A" | sort | uniq -c
💡 Итог

История Git = мощный инструмент:

просмотр изменений
навигация по версиям
поиск ошибок
анализ разработки

📌 База:

git log + фильтры + diff + show = полный контроль над историей
