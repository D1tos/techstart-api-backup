# Просмотр истории и навигация по коммитам
Летопись проекта.Все изменения с момента создания
## Просмотр истории -git log
Отображает список коммитов в обратном хронолическом порядке

Каждая запись содержит:
- идентификатор
- автор и эмейл
- дату и время создания
- сообщение коммита

### Форматы вывода
#### Компактный вывод — одна строка на коммит
$ git log --oneline

#### Показать граф веток
$ git log --graph --oneline

#### Подробная статистика изменений
$ git log --stat

#### Показать изменения в каждом коммите
$ git log -p

### Фильтрация истории
#### Последние N коммитов
$ git log -3

#### Коммиты за период времени
$ git log --since="2 weeks ago"
$ git log --since="2024-03-01" --until="2024-03-15"

#### Коммиты конкретного автора
$ git log --author="Ivan"

#### Коммиты с поиском в сообщении
$ git log --grep="fix"

#### Коммиты, изменившие конкретный файл
$ git log -- README.md

#### Коммиты, изменившие конкретную функцию
$ git log -L :functionName:filename.js

### Настройка вывода формата
#### Пользовательский формат
$ git log --pretty=format:"%h - %an, %ar : %s"
a5f3b8c - Ivan Petrov, 2 hours ago : Add user authentication feature
b7d9e2a - Anna Smirnova, 1 day ago : Initial project structure

##### Полезные плейсхолдеры:
##### %h - сокращённый хеш
##### %H - полный хеш
##### %an - имя автора
##### %ae - email автора
##### %ad - дата (формат зависит от --date=)
##### %ar - относительная дата
##### %s - тема (первая строка сообщения)
##### %b - тело сообщения

## Навигация по истории
### Ссылки на коммиты

SHA-1 хеш — полный (40 символов) или сокращённый (минимум 4)
Имя ветки — указывает на последний коммит ветки
HEAD — указывает на текущий коммит
Тег — именованная ссылка на коммит

### Относительные ссылки
#### Родительские коммиты
HEAD~   или HEAD~1   # Один коммит назад
HEAD~2               # Два коммита назад
HEAD~3               # Три коммита назад

#### Для коммитов слияния
HEAD^   или HEAD^1   # Первый родитель
HEAD^2               # Второй родитель (при слиянии)

#### Комбинирование
HEAD~2^2            # Второй родитель коммита два шага назад
master~3            # Три коммита назад от master

## Просмтор конкретного коммита -git show
Отобрадает подробную информацию о конкретном коммите

Варианты:
### Показать последний коммит
$ git show

### Показать конкретный коммит
$ git show a5f3b8c

### Показать коммит три шага назад
$ git show HEAD~3

### Показать только статистику без изменений
$ git show --stat

### Показать только имена изменённых файлов
$ git show --name-only

## Поиск в истории
### Найти коммит, добавивший или удаливший строку
$ git log -S "function authenticate"

### Найти коммит с изменением количества вхождений
$ git log -G "authenticate.*password"

### Показать кто и когда изменял каждую строку файла
$ git blame filename.js

## Визуализации истории
### Простой граф
$ git log --graph --oneline --all

### Расширенный граф с датами
$ git log --graph --pretty=format:'%h -%d %s (%cr) <%an>' --all

## Сравнений коммитов
git diff

### Изменения между двумя коммитами
$ git diff HEAD~2 HEAD

### Изменения между коммитом и текущим состоянием
$ git diff a5f3b8c

### Только имена изменённых файлов
$ git diff --name-only HEAD~5 HEAD

## Практические сценарии
### Поиск причинв ошибки
просмотр истории изменений файла или функции. найти коммит,внесший проблему

### Анализ развития проекта
статистика по авторам,частота коммитов и измененным файлам = динамика разработки

### Подготовка релиза
просмотр изменений с последнего релиза помогает составить список нововведений
#### Что изменилось с последнего тега
$ git log v1.0..HEAD --oneline

#### Кто больше всего вносил изменений
$ git shortlog -sn

#### Активность по дням недели
$ git log --format="%ad" --date=format:"%A" | sort | uniq -c
