# Установка Git на разные ОС

## Windows
Официальный сайт https://git-scm.com/install/windows
Скачать, установить
Проверить командой git --version

## Linux

### Ubuntu/Debian
sudo apt update && sudo apt install git -y

### Centos/RHEL
sudo yum install git -y

### Arch
sudo pacman -S git

Проверить в терминале командой git --version 

## macOS

Скачать,Установить Homebrew 
https://brew.sh/

brew install git

Проверить в терминале командой git --version
