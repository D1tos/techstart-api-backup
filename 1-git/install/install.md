# Установка Git на разные OC
Процесс установки на различные ОС

## Git Windows
 - Перейти на официальный сайт Git и скачать последнюю версию
 - Запустить скачанный файл и следовать инструкциям установщика.
 (настройки по умолчанию)
 - После установки открыть 'Git Bash' / командную строку, проверить установку косандой
 git --version

## Git Linux
Установить с помощью пакетного менеджера дистрибутива:

### Ubuntu/Debian 
sudo apt update && sudo apt install git -y

### CentOs/RHEL
sudo yum install git -y

### Arch
sudo pacman -S git

После установки проверить установку командной
 git --version

## Git macOS
Установить с помощью пакетного менеджера Homebrew:
 - Установить Homebrew с официального сайта
 - Выполнить в терминале команду:
 brew install git

После установки проверить установку командой
 git --version
